  • ¶

    node-http2 consists of the following components:

    • http.js: public node-http2 API
    var http2        = require('./http');
module.exports = http2;
  • ¶
    • utils.js: common utility functions, like concatenating buffers
    http2.utils      = require('./utils');
  • ¶
    • logging.js: a default logger object and a registry of log formatter functions
    http2.logging    = require('./logging');
  • ¶
    • framer.js: the lowest layer in the stack that transforms between the binary and the JavaScript object representation of HTTP/2 frames
    http2.framer     = require('./framer');
  • ¶
    • compressor.js: compression and decompression of HEADER frames
    http2.compressor = require('./compressor');
  • ¶
    • stream.js: implementation of the HTTP/2 stream concept
    http2.stream     = require('./stream');
  • ¶
    • connection.js: multiplexes streams, manages the identifiers of them and repsonsible for connection level flow control
    http2.connection = require('./connection');
  • ¶
    • endpoint.js: manages other components (framer, compressor, connection, streams) and part of the handshake process
    http2.endpoint   = require('./endpoint');
